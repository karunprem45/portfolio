Save your profile photo in THIS folder as:

    karuniya.jpg

That exact filename is used by:
  - the home-page arch portrait
  - the floating "AI Karuniya" avatar button
  - the assistant chat header

Tips:
  - A square image works best (e.g. 800x800). It will be cropped to a circle/arch automatically.
  - JPG or PNG is fine — if you use PNG, either rename it to karuniya.jpg or
    update the three "assets/karuniya.jpg" references in index.html.
  - If the file is missing, the site gracefully shows a "KP" monogram instead.
